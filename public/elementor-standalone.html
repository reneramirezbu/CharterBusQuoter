Instructions for Elementor Embedding

  1. Copy the HTML container with ID "charter-quote-widget" and class "charter-quote-widget"
  2. In your Elementor page, add an HTML widget
  3. Paste the copied HTML into the widget
  4. Add a link to the quote-widget.css stylesheet in the header of your WordPress site
  5. Add the elementor-widget.js script to the footer of your WordPress site
